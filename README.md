## Requirements

python modules:
- "requests" (install it with pip: `pip install requests`)
- "numpy" (install it with pip: `pip install numpy`)

## Installation

- download or clone the repository

## Run the script

1. open the shell (command line interface)
2. navigate to the directory 
3. run the command:

`python script.py `
